# GESTOR-DE-PERSONAL
GESTION AUTOMATICA DE EMPLEADOS
en esta app vamos a hacer un programa que organice y distribuya


FUNCIONES BASICAS
Registro de empleados (nombre, cargo, contacto)
Control de asistencia (entradas y salidas)
Gestión de horarios y turnos
Base de datos de empleado


GESTION ADMINISTRATIVA
Cálculo de nómina (salarios)
Gestión de vacaciones y permisos
Historial laboral de cada empleado
Generación de reportes (PDF o Excel)


SEGURIDAD Y ACCESOInicio de sesión con roles (admin, empleado, jefe)
Control de acceso a información
codigo qr
App móvil + versión web
